#  Process
 - When a finger nears a button, the IR sensor detects it and sends data to the controller.
 - The controller maps the IR sensor to a floor, shows it on the screen, and moves the elevator.
 - If multiple buttons are pressed, the Arduino resets and prompts to try again on the screen.
 - Components used: Arduino UNO, IR Sensor, IR Sensor, OLED Display, 28BYJ-48 Stepper Motor, ULN2003 Motor Drive.
 - The ElevatorModel/ElevatorDesign shows a design with sensor buttons, a controller, a screen, and a motor.
